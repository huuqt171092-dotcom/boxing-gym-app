Hướng dẫn kết nối Google Sheet (làm 1 lần, mất ~2 phút)
  1. Vào sheets.new để tạo một Google Sheet mới, đặt tên tuỳ ý (ví dụ "Học viên Boxing").
  2. Trong Sheet, chọn menu Tiện ích mở rộng → Apps Script.
  3. Xoá hết code mẫu (myFunction), dán đoạn code bên dưới vào.
  4. Nhấn biểu tượng lưu (đĩa mềm), đặt tên project bất kỳ.
  5. Nhấn nút Triển khai → Triển khai mới. Ở mục "Chọn loại", chọn Ứng dụng web.
  6. Ở "Người thực thi": chọn Tôi (email của bạn). Ở "Ai có quyền truy cập": chọn Bất kỳ ai.
  7. Nhấn Triển khai, cấp quyền truy cập (nếu hiện cảnh báo "Google chưa xác minh app", chọn "Advanced/Nâng cao" → "Đi tới [tên project] (không an toàn)" → Allow). Đây là app bạn tự tạo nên hoàn toàn an toàn.
  8. Copy URL dạng https://script.google.com/macros/s/xxxx/exec và dán vào ô "URL Web App" ở phần Cài đặt phía trên, rồi bấm Lưu.
  9. Mỗi lần thêm/gia hạn học viên, app sẽ tự động gửi 1 dòng dữ liệu vào Sheet "HocVien". Khi xoá học viên trên app, dòng đó sẽ được chuyển sang Sheet "DaXoa" (tự tạo) kèm thời gian xoá, để bạn xem lại lịch sử khi cần.
function doGet(e) {
  return ContentService.createTextOutput('Web App đang hoạt động. Hãy dùng URL này trong phần Cài đặt của app Boxing.');
}

function doPost(e) {
  var ss = SpreadsheetApp.getActiveSpreadsheet();
  var sheet = ss.getSheetByName('HocVien') || ss.insertSheet('HocVien');
  var trashSheet = ss.getSheetByName('DaXoa') || ss.insertSheet('DaXoa');

  var headers = ['ID','Họ tên','SĐT','Ngày bắt đầu','Gói tập','Số ngày','Ngày hết hạn','Ghi chú','Loại','Thời gian gửi'];

  if (sheet.getLastRow() === 0) {
    sheet.appendRow(headers);
  }
  if (trashSheet.getLastRow() === 0) {
    trashSheet.appendRow(headers.concat(['Thời gian xoá']));
  }

  var data = JSON.parse(e.postData.contents);

  if (data.action === 'xoa') {
    var values = sheet.getDataRange().getValues();
    for (var i = values.length - 1; i >= 1; i--) {
      if (String(values[i][0]) === String(data.id)) {
        var rowData = values[i].concat([new Date()]);
        trashSheet.appendRow(rowData);
        sheet.deleteRow(i + 1);
      }
    }
    return ContentService.createTextOutput(JSON.stringify({status:'ok', action:'deleted'}))
      .setMimeType(ContentService.MimeType.JSON);
  }

  sheet.appendRow([
    data.id,
    data.name,
    data.phone,
    data.startDate,
    data.packageLabel,
    data.durationDays,
    data.endDate,
    data.note || '',
    data.action || 'dang_ky',
    new Date()
  ]);

  return ContentService.createTextOutput(JSON.stringify({status:'ok'}))
    .setMimeType(ContentService.MimeType.JSON);
}

Lưu ý: trình duyệt gọi URL này ở chế độ "no-cors" nên app không đọc được phản hồi chi tiết từ Google — nếu URL đúng và đã triển khai với quyền "Bất kỳ ai", dữ liệu vẫn sẽ được ghi vào Sheet bình thường, cứ kiểm tra lại Sheet để xác nhận.
